## Migration Guide

### For Users
1. Install kiro-protocols power via Kiro IDE Powers panel
2. Add repository: https://github.com/Theadd/kiro-agents
3. Install "Kiro Protocols" power
4. Use protocols on-demand with kiroPowers tool

### For Developers
1. Modify source files in src/core/protocols/
2. Run: bun run build:powers
3. Test locally in Kiro IDE
4. Submit PR (without powers/*/steering/ changes)
5. Maintainers regenerate on merge
